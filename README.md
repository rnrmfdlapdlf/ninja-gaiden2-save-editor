브라우저에서 실행되는 단일 HTML 기반의 **Ninja Gaiden 2 Save Editor**입니다. 세이브 파일을 화면에 드래그 앤 드롭하거나 파일 선택 버튼으로 열고, 값을 수정한 뒤 체크섬이 갱신된 파일을 다운로드할 수 있습니다.

## 실행 방법

1. 이 저장소의 `index.html` 파일을 다운로드합니다.
2. 브라우저에서 `index.html`을 엽니다.
3. 세이브 파일을 화면에 드래그 앤 드롭하거나 **열기 / Open** 버튼으로 선택합니다.
4. 원하는 값을 수정합니다.
5. **다운로드 / Download** 버튼을 눌러 수정된 세이브 파일을 저장합니다.

# Ninja Gaiden 2 Save Editor

A single-file, browser-based **Ninja Gaiden 2 Save Editor**. Open a save file by dragging it onto the page or by using the file picker, edit the values, then download a checksum-fixed copy.

## How to Use

1. Download `index.html` from this repository.
2. Open `index.html` in your browser.
3. Drag and drop a save file onto the page, or select one with **Open**.
4. Edit the values you want to change.
5. Click **Download** to save the edited file.

## Supported Formats

- Xbox 360
- Raw payload
- Xenia

## Editable Values

- Yellow Essence
- Karma
- Health
- Ninpo Slots
- Incendiary Shuriken
- Fiends Bane Bow Arrows
- Herb Of Spiritual Life
- Grains Of Spiritual Life
- Devil Way Mushroom
- Life Of The Gods
- Life Of The Thousand Gods
- Jewel Of The Demon Seal
- Spirit Of The Devils
- Talisman Of Rebirth
- Muramasa's Omusabi

## Features

- Drag-and-drop save loading
- Automatic Xbox 360 / raw payload / Xenia format detection
- Automatic checksum recalculation after editing
- Reset values to their original state
- Optional EXE-compatible random field update
- Runs as a standalone HTML file with no installation required

## Warning

Always back up your original save file before editing. Invalid values or unsupported save formats may prevent the save file from loading correctly.
